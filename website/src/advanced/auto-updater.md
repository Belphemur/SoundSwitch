# Auto Updater

SoundSwitch checks for updates every 24 hours via the GitHub API.

## Update Modes

You can configure how SoundSwitch handles updates in the **General** settings tab:

| Mode | Behavior |
|------|----------|
| **Silent / Install updates automatically** | Downloads and applies updates in the background without prompting. |
| **Notify / Notify me when updates are available** | Shows a notification when an update is available, allowing you to choose when to update. |
| **Never / Never check for updates** | Disables automatic update checking entirely. |

### Include Beta Versions

When enabled, SoundSwitch considers pre-release (beta) versions as available updates. This is useful for testing upcoming features but may introduce instability.
