# Communication Devices

SoundSwitch can change the **Default Communication Device** when the "Switch Default Communication Device" option is enabled on the **General** settings tab.

## Why Does This Matter?

Windows differentiates between two types of audio devices:

- **Multimedia device** — The standard default playback/recording device used by most applications.
- **Communication device** — A separate default used by communication apps like Discord, Teams, Skype, and Zoom.

By default, SoundSwitch only changes the multimedia device when you switch. If you want communication apps to follow the same device changes, enable **Switch Default Communication Device** in the General settings. When enabled, switching the default playback device will also switch the Windows Default Communication Device to the same device.

## How to Enable

1. Open SoundSwitch **Settings** (right-click the tray icon → **Settings**).
2. Go to the **General** tab.
3. Check the **Switch Default Communication Device** checkbox.
4. Click **Close**.

This ensures that when you switch to your headphones, both general audio and voice chat applications use the headphones.
