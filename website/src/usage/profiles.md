# Profiles

Profiles allow you to automatically switch to specific audio devices when certain conditions are met.

## Triggers

Profiles support multiple trigger types:

- **Hotkey Triggers**: Switch devices using custom key combinations.
- **Application Triggers**: Automatically switch when specific applications gain focus.
- **Window Triggers**: Switch based on window titles.
- **Steam Big Picture**: Special profile for Steam Big Picture mode.
- **UWP App Triggers**: Support for Universal Windows Platform apps.
- **Startup Triggers**: Activate when SoundSwitch starts.
- **Device Changed Triggers**: Maintain specific device configurations.
- **Tray Menu Triggers**: Accessible from the system tray context menu.

## Advanced Features

- Multi-Device Support
- Smart Device Restoration
- Foreground App Switching
- Default Device Control
- Notification Control
- Device Validation

## Hotkey Cycling

When multiple profiles share the same hotkey, SoundSwitch automatically cycles through them.
