# Hotkeys

SoundSwitch uses hotkeys to quickly switch between audio devices. Hotkeys can be configured on the **Playback** and **Recording** tabs of the Settings window, as well as within individual profiles.

## Default Hotkeys

| Action | Default Hotkey |
|--------|---------------|
| **Switch playback device** | `Ctrl` + `Alt` + `F11` |
| **Switch recording device** | `Ctrl` + `Alt` + `F7` |
| **Toggle microphone mute** | `Ctrl` + `Alt` + `M` |

## Where to Configure Hotkeys

- **Playback switch hotkey** — Set on the **Playback** tab in Settings.
- **Recording switch and mute hotkeys** — Set on the **Recording** tab in Settings.
- **Profile hotkeys** — Assign hotkeys to individual profiles on the **Profiles** tab. Multiple profiles can share a hotkey to create a cycling group.

You can enable or disable each hotkey independently using the **Hotkey enabled** checkbox next to it.
