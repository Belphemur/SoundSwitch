# Hotkeys

SoundSwitch uses hotkeys to quickly switch between audio devices.

## Default Hotkeys

- **Playback devices**: `Ctrl` + `Alt` + `F11`
- **Recording devices**: `Ctrl` + `Alt` + `F7`
- **Mute microphone**: `Ctrl` + `Alt` + `M`

## Customization

You can customize these hotkeys in the SoundSwitch settings.
