# Notifications

The **Notifications** tab lets you choose how SoundSwitch alerts you when an audio device is switched or when your microphone is muted or unmuted.

![Notifications](/images/Notifications.png)

## Notification Type

The **Notification Type** field shows the current display mode for device switch notifications. You can click the **"..."** button to open advanced options and choose between different notification styles. The **Switch device** dropdown lets you pick the notification style for device changes; options include a banner notification, a Windows toast notification, a sound chime, or no notification.

## Custom Sound File

On the right side of the top row, click **Select...** to choose an audio file that SoundSwitch will play when a device is switched.

## Banner Options

When using banner notifications, you can customize their behavior:

- **Always use primary screen** — Forces the banner to display on your primary monitor instead of the monitor with the currently focused window.
- **Only one banner** — When enabled, only one banner is shown at a time. If multiple events occur, they queue rather than overlap.
- **On-screen time** — How long the banner stays visible (default 3 seconds).
- **Opacity** — How transparent the banner is (0–100%).
- **Display Info** — How much detail to show on the banner. Options include device name only, icon only, or full display with both.

## Position

Use the 3x3 grid to choose where on the screen the banner appears (top-left, top-right, top-center, middle-left, middle-right, middle-center, bottom-left, bottom-right, bottom-center). A slider below the grid lets you fine-tune the offset from the edge.

## Microphone Mute

Controls notifications when the microphone mute toggle is activated:

- **Microphone On** — Notification style when the microphone is unmuted.
- **Microphone Off** — Notification style when the microphone is muted.

Options include **None** (no notification), **Banner**, or **Sound** for each state.
