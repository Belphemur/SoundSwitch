# General

The **General** tab contains the core application settings that control how SoundSwitch starts, updates, and interacts with Windows.

## Startup

- **Start automatically with Windows** — Launch SoundSwitch in the background every time you log in.

## Updates

- **Update settings** — Choose how SoundSwitch handles new versions:
  - Notify you when an update is available.
  - Install updates automatically.
  - Include beta versions in the update check.

## Language

- **Language** — Change the display language of the application. A restart is required for the change to take effect.

## Audio Options

- **Communication device** — Also switch the Windows Default Communication Device when cycling devices. By default, SoundSwitch only changes the multimedia device.
- **Keep volume levels across playback devices** — Maintain the same volume level when switching between playback devices.

## System Tray

- **Keep system tray icon** — Always show the SoundSwitch icon in the system tray.
- **Double-click action** — Choose what happens when you double-click the tray icon (e.g., cycle playback or recording devices).
- **Tooltip on hover** — Show the current playback or recording device name when hovering over the tray icon.

## Quick Menu

- **Quick Menu** — Show a visual selector (similar to the Windows language menu) when you press a hotkey, allowing you to pick a device instead of cycling blindly.

## Troubleshooting

- **Reset audio devices** — Clear the current device selection and start fresh.
- **Export log files** — Save application logs to help diagnose issues.
