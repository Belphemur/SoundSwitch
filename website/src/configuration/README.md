# Configuration

SoundSwitch's settings are organized into several tabs. Open the settings by right-clicking the system tray icon and selecting **Settings**.

- **[Playback](playback.md)** — Choose which audio output devices to cycle through and set the playback switch hotkey.
- **[Recording](recording.md)** — Choose which audio input devices to cycle through and set the recording switch and mute hotkeys.
- **[General](general.md)** — Control startup behavior, updates, language, system tray options, and troubleshooting.
- **[Notifications](notifications.md)** — Customize how SoundSwitch alerts you when devices change.
