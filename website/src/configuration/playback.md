# Playback

The **Playback** tab is where you tell SoundSwitch which audio output devices (speakers, headphones, etc.) it should cycle through.

## Selecting Devices

The main area lists all playback devices detected by Windows, grouped by their connection status:

- **Connected** — Devices that are currently available.

To include a device in the rotation, check the box next to its name. SoundSwitch will only switch between the devices you have checked. If you have selected **"Only selected audio devices"** on the General tab, only these checked devices will be cycled through.

## Hotkey Settings

At the bottom of the tab, you can configure the global hotkey used to cycle through your selected playback devices:

- **Switch device** — Click the hotkey field and press the key combination you want to use (the default is `Ctrl + Alt + F11`).
- **Hotkey enabled** — Check this box to activate the hotkey. Uncheck it to disable switching.
