# How do I manually check for an update?

Right-click the **SoundSwitch tray icon** and choose **Update**. SoundSwitch will check for a newer version and notify you if one is available.

![Update menu in the tray](/images/faq/tray-update-menu.png)

This works even if your update preference is set to **Never check for updates**.
