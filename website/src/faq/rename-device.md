# How do I rename a device or change its icon?

## Why rename a device?

When two devices share the same name (for example, two identical monitors with built-in speakers), Windows can't tell them apart, and neither can SoundSwitch. Renaming one of them solves the problem.

## How to rename

1. Open **SoundSwitch** → **Playback Devices** (or **Recording Devices**).
2. **Double-click** the device you want to rename.
3. Type the new name and choose an icon if you'd like.
4. Click **Save**.

![Renaming a device](/images/faq/rename-device.gif)

After renaming, re-tick the device in SoundSwitch so it's included in the cycle.

## What icon size should I use?

For best results, provide an **`.ico` file containing 8×8 and 16×16 sizes**:

- **8×8** is used for the system tray.
- **16×16** is used inside the notification banner.

Windows requires the file to be a real `.ico` (not a renamed `.png`).
