# Getting Started

## Requirements

- Operating System: Windows 7 or newer

## Installation

Download the latest release from the [SoundSwitch website](https://soundswitch.aaflalo.me) and run the installer.

## Basic Usage

After installation, SoundSwitch runs in the background. Use the default hotkeys to switch devices:

- **Playback devices**: `Ctrl` + `Alt` + `F11`
- **Recording devices**: `Ctrl` + `Alt` + `F7`
- **Mute microphone**: `Ctrl` + `Alt` + `M`

## System Tray Actions

Right-click the SoundSwitch icon in the system tray to access settings and device selection. You can also double-click the icon to cycle through playback devices.
