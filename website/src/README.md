## Quick Start

Get up and running in under a minute:

1. **Install** SoundSwitch using the download button above.
2. **Right-click** the SoundSwitch tray icon and open **Settings**.
3. On the **Playback** and **Recording** tabs, check the devices you want to cycle through.
4. Use the default hotkeys to switch instantly:
   - `Ctrl` + `Alt` + `F11` — cycle playback devices
   - `Ctrl` + `Alt` + `F7` — cycle recording devices
   - `Ctrl` + `Alt` + `M` — toggle microphone mute

Need more detail? Head over to the [full Getting Started guide](./getting-started.md).
